Random Name
========

A class project for CS480 at the University of Idaho to create an Android Application to select a random name from a set of lists

Android Specifics
----------
- Minimum Android API: 7 (2.1 - Eclair)
- Target Android API: 16 (4.1 - Jelly Bean)
